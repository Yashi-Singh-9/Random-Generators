---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Yashi-Singh-9/Random-Generators.git
cd random-generators
````

### Run a Generator

Each generator is self-contained. You can open any `index.html` file in your browser:

```bash
cd "Random Book Generator"
start index.html  # or open manually in your browser
```

## 🎨 Preprocessor Installation

Some projects use CSS preprocessors. Use the appropriate tools below:

### SCSS / SASS

```bash
npm install -g sass

# Compile SCSS
sass style.scss style.css

# Compile SASS (indented syntax)
sass style.sass style.css
```

### Stylus

```bash
npm install -g stylus
stylus style.styl
```

### Less

```bash
npm install -g less
lessc style.less style.css
```
